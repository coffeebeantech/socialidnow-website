Personalização

Customize a aparência e experiência do seu website com os dados de perfil social do usuário. Recomende produtos de acordo com as preferências do usuário. Mostre a propaganda certa para o usuário.

Personalização de Website

Aumente o engajamento com o usuário usando seus dados de perfil.

Por exemplo, use o nome e a foto capturados da rede social e se comunique de forma mais pessoal com o usuário. Use dados de localização para oferecer experiências baseadas em geolocalização. Use o grafo social do usuário para engajar amigos, aumentando a conversão.

Recomendação de Produtos

Use o perfil social do usuário para encontrar os melhores produtos e informações para recomendar a ele.

Por exemplo, use os likes do Facebook para traçar categorias de preferência do usuário e alimentar um sistema de recomendação. Ou use a data de aniversário do usuário e de seus amigos para recomendar presentes.

Email Marketing Personalizado

Monte diferentes conteúdos de email a partir das preferências sociais do usuário.

Em vez de enviar um mesmo email para todos seus usuários, envie emails personalizados de acordo com as necessidades de cada usuário, aumentando assim as taxas de conversão e cliques.
