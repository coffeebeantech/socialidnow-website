Integração

Possuimos APIs completas para integração, tanto em server-side (RESTful) quanto front-side (Javascript).

Server-side APIs

Integre com nossos serviços web em RESTful. Temos APIs para o Social Login, Social Webform e Marketing Database.

Realize segmentações em tempo real. Compartilhe posts no Facebook. Obtenha dados do perfil social do usuário para personalização.

Front-side APIs

Implemente experiências "single-page" usando nossas APIs Javascript, desta forma não é preciso redirecionar o usuário para diferentes páginas, o que torna a navegação muito mais rápida e amigável.
